KINMILL
----

**Dependencies**

* Vagrant
* Vagrant Plugins:
    Vagrant berkshelf
    Vagrant omnibus
* Berkshelf

**Stack**

* Ruby 2.1.2
* Rails 4.1.1
* Postgresql

**Development**

To begin development, just run <tt>vagrant up</tt>.

If you need to make changes to the server, update the Berksfile with the
appropriate cookbooks.
